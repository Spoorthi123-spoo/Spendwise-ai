# SpendWise AI

SpendWise AI is an AI subscription audit platform that helps startups and teams optimize AI software spending.

## Features

- AI Spend Analysis
- Smart Recommendations
- Annual Savings Estimation
- Email Capture
- Shareable Audit URLs
- AI Summary Recommendations
- Mobile Responsive Design

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL

## Getting Started

Install dependencies:

npm install

Run development server:

npm run dev

Open:

http://localhost:3001
